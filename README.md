# Бот-помощник для экзамена по операционным системам в Telegram

Умный бот, созданный для помощи в изучении и повторении материала по дисциплине "Операционные системы". В основе работы лежат возможности искусственного интеллекта от Yandex Cloud.

## Основные возможности

*   **Интеллектуальные текстовые ответы**: получайте развернутые и структурированные ответы на вопросы по предмету, сгенерированные моделью YandexGPT.
*   **Чтение вопросов с изображений**: просто отправьте боту фотографию или скриншот с текстом вопроса, и система распознает его с помощью Yandex Vision, чтобы дать ответ.
*   **Простой интерфейс Telegram**: поддерживаются базовые команды `/start` и `/help` для удобного взаимодействия.
*   **Устойчивость к ошибкам**: бот корректно обрабатывает некорректные запросы и информирует пользователя о возникших проблемах.

## Как это работает: схема взаимодействия

1.  Пользователь отправляет сообщение или фото в Telegram.
2.  Запрос поступает на серверную функцию в Yandex Cloud Functions.
3.  В зависимости от типа запроса, функция обращается либо к YandexGPT (для текста), либо к Yandex Vision (для извлечения текста с фото).
4.  Полученный ответ отправляется обратно пользователю в чат.

**Задействованные сервисы Yandex Cloud:**
*   **Cloud Functions**: исполняемая среда для кода бота.
*   **YandexGPT (LLM)**: генерация содержательных ответов на вопросы.
*   **Yandex Vision**: оптическое распознавание текста на загруженных изображениях.
*   **Object Storage**: надежное хранение системных инструкций и промптов для AI.
*   **IAM (Identity and Access Management)**: управление правами доступа и API-ключами для безопасной работы сервисов.

## Развертывание инфраструктуры

Управление всеми необходимыми ресурсами в Yandex Cloud осуществляется с помощью Terraform.

| Действие | Команда | Описание |
| :--- | :--- | :--- |
| **Инициализация** | `terraform init` | Подготавливает рабочую директорию, загружая провайдеры и модули. |
| **Создание** | `terraform apply -auto-approve` | Разворачивает всю инфраструктуру (функцию, сервисный аккаунт, бакет и т.д.). |
| **Удаление** | `terraform destroy -auto-approve` | Полностью очищает все созданные ресурсы в облаке. |
